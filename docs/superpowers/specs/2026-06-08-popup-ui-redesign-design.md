# 界面优化设计文档

**日期**: 2026-06-08  
**范围**: Popup 设置面板 + 页面内翻译元素 + 交互动画

## 设计目标

将界面从基础的蓝色扁平风格升级为现代、精致、有层次感的视觉体验，同时保持所有现有功能不变。

## 配色方案

基于图标主色调，使用靛紫渐变色系：

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主渐变起始 | `#818CF8` | 浅紫色 |
| 主色 | `#6366F1` | 靛紫色，图标核心色 |
| 主渐变结束 | `#4F46E5` | 深紫色 |
| 辅助色 1 | `#C7D2FE` | 浅紫，用于边框装饰 |
| 辅助色 2 | `#A5B4FC` | 中浅紫 |
| 辅助色 3 | `#E0E7FF` | 极浅紫，用于背景/加载态 |
| 背景色 | `#F8F9FC` | 灰白底色 |
| 卡片背景 | `#FFFFFF` | 纯白 |

### 渐变定义

```css
/* 主渐变 - 用于头部、主按钮 */
background: linear-gradient(135deg, #818CF8 0%, #6366F1 50%, #4F46E5 100%);

/* 进度条渐变 */
background: linear-gradient(90deg, #818CF8, #6366F1);
```

## 圆角与阴影规范

| 元素 | 圆角 | 阴影 |
|------|------|------|
| 按钮 | 5-6px | `0 2px 8px rgba(99,102,241,0.25)` |
| 输入框 | 5px | 无 |
| 卡片 | 6px | `0 1px 3px rgba(0,0,0,0.05)` |
| 进度条 | 8px | `0 4px 16px rgba(99,102,241,0.1)` |

## Popup 设计

### 整体结构

```
┌─────────────────────────────┐
│  HEADER (渐变背景)           │
│  ┌───────────────────────┐  │
│  │ 图标 + 标题 + 副标题   │  │
│  │ 连接状态胶囊           │  │
│  └───────────────────────┘  │
├─────────────────────────────┤
│  快捷操作 (两个按钮)         │
│  [翻译页面] [智能翻译]       │
├─────────────────────────────┤
│  Tab 标签栏                  │
│  [API设置] [触发方式] [缓存] │
├─────────────────────────────┤
│  Tab 内容区 (白色卡片)       │
│  ┌───────────────────────┐  │
│  │  表单内容              │  │
│  └───────────────────────┘  │
├─────────────────────────────┤
│  底部操作栏                  │
│  [测试连接] [保存配置]       │
├─────────────────────────────┤
│  FOOTER (快捷键提示)         │
└─────────────────────────────┘
```

### Header

- **背景**: 靛紫渐变 `linear-gradient(135deg, #818CF8 0%, #6366F1 50%, #4F46E5 100%)`
- **Logo**: 32x32 半透明白色背景容器，内含简化版翻译图标（SVG）
- **标题**: "Ollama Translator"，15px，白色，font-weight: 600
- **副标题**: "英译汉翻译助手"，10px，白色 85% 透明度
- **连接状态**: 胶囊样式，半透明白色背景 `rgba(255,255,255,0.18)`
  - 圆点指示器（已连接=绿色 `#a5f3a0`，未连接=红色）
  - 显示当前模型名

### 快捷操作区

- 两个按钮并排，flex 布局，gap: 8px
- **翻译页面**: 渐变背景 + 白色文字 + 阴影
- **智能翻译**: 白色背景 + 紫色文字 + 紫色边框 `border: 1px solid #C7D2FE`

### Tab 标签栏

- 三个标签：API 设置 / 触发方式 / 缓存
- 选中标签：紫色文字 + 底部 2px 紫色指示条
- 未选中标签：灰色文字 `#999`
- 底部 1px 分隔线 `#E8EAF0`

### Tab 内容区

- 背景色：`#F8F9FC`
- 内容包裹在白色卡片中
- 卡片内边距：14px
- 卡片阴影：`0 1px 3px rgba(0,0,0,0.05)`

#### API 类型切换

- 从下拉框改为按钮式切换
- 两个按钮并排，选中态为渐变背景，未选中为灰色背景 `#F5F5F7`

#### 表单字段

- Label：11px，灰色 `#888`，font-weight: 500
- Input：1px 边框 `#E0E3EF`，圆角 5px，背景 `#FCFCFE`
- 聚焦态：边框变为 `#C7D2FE`

#### 模型选择

- 输入框 + 刷新按钮并排
- 刷新按钮：32px 宽，灰色背景，圆角 5px

### 底部操作栏

- 两个按钮：测试连接（次要）/ 保存配置（主要）
- 样式同快捷操作区

### Footer

- 白色背景
- 显示快捷键信息，10px，灰色 `#BBB`
- 顶部 1px 分隔线 `#F0F0F0`

## 页面内翻译元素设计

### 译文样式

- **左侧装饰线**: 3px 紫色竖线 `border-left: 3px solid #C7D2FE`
- **译文颜色**: 主题紫 `#6366F1`
- **字号**: 13px（略小于原文）
- **行高**: 1.5

### 段落翻译按钮

- 悬停时显示在段落右侧
- 尺寸：24x24px
- 背景：靛紫渐变
- 文字："译"，白色，11px
- 阴影：`0 2px 8px rgba(99,102,241,0.3)`

### 翻译进度条

- 位置：右上角固定
- 白色卡片，圆角 8px
- 阴影：`0 4px 16px rgba(99,102,241,0.1)`
- 内容：
  - 左侧：圆形渐变图标（内含旋转动画）
  - 中间："翻译中..." 文字 + 渐变进度条
  - 右侧：百分比数字 + 关闭按钮

### 加载状态

- 旋转圆环动画
- 边框色：`#C7D2FE`
- 旋转部分：`#6366F1`
- 文字："翻译中..."，颜色 `#A5B4FC`

### 错误状态

- 左侧装饰线：红色 `#FECACA`
- 文字：红色 `#EF4444`
- 可点击重试

## 交互动画

### 按钮悬停

- 背景色略微变亮
- 可选：轻微上移 `transform: translateY(-1px)`
- 过渡：`transition: all 0.2s ease`

### Toast 提示

- 渐变背景
- 从底部滑入
- 圆角 6px
- 阴影增强

### Tab 切换

- 指示条滑动动画
- 内容区淡入

## 实现要点

### Popup HTML 结构变化

1. Header 区域：新增 logo 容器和连接状态胶囊
2. 快捷操作区：从 section 内部移到 header 下方
3. 新增 Tab 标签栏
4. 各 section 用 Tab 内容包裹
5. API 类型：从 `<select>` 改为按钮组
6. Footer：显示快捷键信息

### Popup CSS 主要变化

1. 全局配色变量替换
2. 新增渐变背景定义
3. 卡片样式
4. Tab 样式
5. 按钮悬停效果
6. 连接状态胶囊样式

### Content CSS 变化

1. 译文左侧装饰线
2. 段落翻译按钮渐变
3. 进度条卡片样式
4. 加载动画优化

### JavaScript 变化

1. Tab 切换逻辑
2. API 类型按钮组逻辑（替代 select）
3. Toast 动画优化（可选）

## 不变的部分

- 所有功能逻辑
- 消息协议
- Storage 结构
- API 调用方式
- 快捷键绑定
